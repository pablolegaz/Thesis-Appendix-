# BFM installation notes

The experiments use the Belief Function Machine (BFM) in MATLAB.

The scripts assume that BFM functions such as `uil2bm`, `condiembed`, `keepbel`, `solve`, and `showbel` are available on the MATLAB path.

Suggested local setup:

1. Install MATLAB.
2. Obtain/install BFM separately.
3. Add the BFM directory to the MATLAB path.
4. Set the repository root as the working directory.
5. Run the export scripts in `matlab/export_scripts/`.

The BFM package itself is not redistributed here unless its licence allows it.
