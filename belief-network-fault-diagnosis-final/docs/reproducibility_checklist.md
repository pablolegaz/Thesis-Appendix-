# Reproducibility checklist

Before submitting or linking the repository:

- [ ] README explains the thesis and repository structure.
- [ ] MATLAB export scripts are included.
- [ ] UIL source models are included.
- [ ] Raw CSV results are included.
- [ ] Diary logs are included.
- [ ] The tutor overview notebook opens correctly.
- [ ] No private local file paths remain in the main documentation.
- [ ] Thesis tables match the CSV files.
- [ ] External BFM package licensing is not violated.
