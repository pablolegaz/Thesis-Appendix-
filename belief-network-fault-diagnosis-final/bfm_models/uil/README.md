# UIL model files

These `.txt` files are BFM Unified Input Language (UIL) files. They are the most readable source representation of the belief-network models.

A UIL file contains:

- `DEFINE VARIABLE`: model variables and their states.
- `DEFINE RELATION`: ordinary belief-function relations.
- `DEFINE CONDITIONAL RELATION`: conditional belief relations.
- `SET VALUATION`: mass assignments, deterministic constraints, or vacuous ignorance.

The L1, L2, and L4 files are temporary scenario-specific models generated by the MATLAB scripts from the L0 base UIL. They are included as examples of the generated model variants. The MATLAB scripts are the authoritative source for how each variant is produced.
